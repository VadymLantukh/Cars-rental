# Project: AutoRental :blue_car:


**AutoRental** - This site is developed on React for renting and reviewing car catalogs. [View project](https://cars-rental-tau.vercel.app/)

**API** - I used the [mockAPI](https://mockapi.io/projects), created a project and resource data with information about the machines. [documentation mockAPI](https://github.com/mockapi-io/docs/wiki)

### Project structure
The structure consists of pages: home, favorite, catalog, and components, and the site also uses Redux-Persist technology to store favorite machines after reloading the page.
